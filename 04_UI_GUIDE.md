# 인생여전 UI Guide

우선 디자인 철학은 [10_UI_DESIGN_PHILOSOPHY.md](./10_UI_DESIGN_PHILOSOPHY.md)를 따른다. 모든 참가자 화면은 Toss Securities 70%, Apple Stocks 20%, Game UI 10% 비율의 미니멀한 금융 앱 경험을 기준으로 판단한다.

## 1. 전체 UI 방향

전체 디자인은 토스증권, 애플 주식 앱처럼 만든다.

키워드:

- 모바일 우선
- 큰 숫자
- 카드형 UI
- 깨끗한 여백
- 직관적인 투자 버튼
- 상승/하락 색상 구분
- 실시간 그래프 중심
- 관리자 화면은 표와 대시보드 중심
- 전광판은 멀리서도 보이는 큰 글씨 중심

색상:

- 배경: `#F5F6F8` 또는 밝은 회색
- 카드: 흰색
- 텍스트: 검정, 짙은 회색
- 상승: 빨간색 계열
- 하락: 파란색 계열
- 버튼: 검정 또는 진한 남색

## 2. 페이지 구조

- `/participant`
- `/admin`
- `/display`

## 3. 참가자 페이지

목적:

참가자는 자기 휴대폰으로 접속한다. 참가자 페이지는 내 자산, 내 보유 투자, 실시간 그래프, 투자하기에 집중한다. 전체 회원 순위, 실명 목록, 관리자 기능은 참가자 페이지에 노출하지 않는다.

화면 순서:

1. 자산 요약 카드
2. 상태 안내
3. 실시간 주식 그래프
4. 내 보유 투자
5. 투자 가능한 기업 카드
6. 공지/뉴스

자산 요약 카드 표시 정보:

- 총자산
- 수익률
- 보유 현금
- 투자 평가액
- 닉네임
- 소속 회사

실시간 그래프:

- Recharts `LineChart`
- 기업별 `currentValue` 표시
- 10초 또는 1분 단위 갱신
- 그래프 아래 기업별 현재 가치와 변동률 표시

보유 투자 카드:

- 기업명
- 내 투자금
- 현재 평가액
- 수익률
- 현재 기업 가치
- 변동률

기업 투자 카드:

- 기업명
- 현재 기업 가치
- 변동률
- 내 투자금
- 투자하기 버튼

투자 모달:

- 보유 현금
- 투자 금액 입력
- 빠른 금액 버튼
- 전액 버튼
- 보유 현금보다 큰 금액 입력 불가
- 0 이하 입력 불가
- 투자 마감 상태에서는 버튼 비활성화
- 투자 완료 후 토스트 표시

공지/뉴스:

- 공지는 상단 토스트 또는 화면 중앙 모달
- 뉴스는 상단 배너 또는 카드

## 4. 관리자 페이지

목적:

관리자는 노트북으로 접속한다. 관리자 페이지는 행사 운영 대시보드다.

구성:

- 상단 고정 상태바
- 실시간 통계 카드
- 운영 버튼
- 뉴스/공지 발송
- 개인 랭킹
- 기업 랭킹
- 최근 투자 로그
- 회원 관리 테이블

상단 상태바:

- 현재 연차
- 현재 상태
- 남은 시간
- 접속 인원

운영 버튼:

- 연봉 지급
- 투자 시작
- 투자 마감
- 정산 확정
- 다음 연차
- 게임 종료
- 일시정지
- 재개

뉴스/공지:

- 뉴스 제목
- 뉴스 내용
- 발송 버튼
- 공지 내용
- 빠른 공지 버튼

빠른 공지 예시:

- 투자가 1분 후 마감됩니다!
- 뉴스가 공개되었습니다.
- 모든 참가자는 결과를 확인해주세요.
- 투자가 종료되었습니다.
- 다음 연차가 시작됩니다.

## 5. 전광판 페이지

목적:

전광판은 빔프로젝터/TV에 띄우는 보기 전용 화면이다. 조작 버튼은 없다.

구성:

- 현재 연차/상태
- 남은 시간
- 기업 TOP5
- 개인 TOP5
- 최신 뉴스
- 최신 공지
- 최근 투자 로그
- 마지막 10초 카운트다운
- 종료 후 엔딩 화면

종료 화면:

- “투자가 종료되었습니다.”
- “최종 정산 중...”
- 기업 1등 공개
- 개인 1등 공개

## 6. 반응형 기준

참가자 페이지:

- 모바일 최우선
- 360px~430px 폭 기준
- 한 손 조작 가능

관리자 페이지:

- 노트북/태블릿 기준
- 표와 카드가 한눈에 보여야 함

전광판:

- 16:9 화면 기준
- 멀리서도 글자가 보여야 함
- 버튼 없이 보기 전용

## 7. 구현 주의사항

- 참가자 페이지에는 관리자 기능을 노출하지 않는다.
- 실명은 관리자 페이지에서만 표시한다.
- 참가자에게는 닉네임 중심으로 표시한다.
- 모든 투자 데이터는 DB에 저장한다.
- 30~40명이 같은 데이터를 공유해야 한다.
- 투자 마감 상태에서는 투자 버튼을 반드시 비활성화한다.
- 일시정지 상태에서는 타이머, 그래프, 투자가 모두 멈춰야 한다.
