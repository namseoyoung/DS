# 인생여전 User Flow

## 1. 전체 사용자 흐름

### 참가자

로그인
→ 닉네임/실명 확인
→ 참가자 페이지 진입
→ 본인 자산 확인
→ 현재 연차/상태 확인
→ 투자 가능 상태일 때 기업에 투자
→ 공지/뉴스 확인
→ 정산 결과 확인
→ 4년차 실시간 투자 참여
→ 최종 결과 확인

### 관리자

로그인
→ 관리자 대시보드 진입
→ 현재 연차/상태 확인
→ 연봉 지급
→ 투자 시작
→ 뉴스/공지 발송
→ 투자 마감
→ 정산 확정
→ 다음 연차 진행
→ 4년차 실시간 라운드 시작
→ 최종 정산
→ 엔딩 공개

### 전광판

`/display` 접속
→ 현재 연차/상태 표시
→ 남은 시간 표시
→ 기업 순위 표시
→ 개인 TOP5 표시
→ 최신 뉴스 표시
→ 마지막 10초 카운트다운
→ 최종 기업 1등 공개
→ 개인 1등 공개

## 2. 연차별 흐름

### 1년차

관리자: 연봉 지급
→ 참가자: 보유 현금 증가
→ 관리자: 투자 시작
→ 참가자: 기업 선택 후 투자
→ 관리자: 투자 마감
→ 관리자: 기업별 변동률 입력
→ 관리자: 정산 확정
→ 참가자: 자산 변동 확인
→ 전광판: 결과 표시

### 2년차

관리자: 연봉 지급
→ 관리자: 정보/뉴스 공개
→ 참가자: 뉴스 확인
→ 관리자: 투자 시작
→ 참가자: 투자
→ 관리자: 투자 마감
→ 관리자: 정산 확정
→ 참가자: 결과 확인

### 3년차

2년차와 동일.

### 4년차

관리자: 4년차 실시간 라운드 시작
→ 전광판: 주식시장 개장 화면 표시
→ 참가자: 실시간 그래프 확인
→ 참가자: 기업에 투자
→ 시스템: 10초 또는 1분 단위로 기업 가치 갱신
→ 관리자: 뉴스/공지 발송
→ 전광판: 기업 순위/개인 TOP5 실시간 표시
→ 시스템: 1분 전 자동 공지
→ 시스템: 마지막 10초 카운트다운
→ 시스템: 투자 자동 마감
→ 시스템: 최종 정산
→ 전광판: 기업 1등 공개
→ 전광판: 개인 1등 공개

## 3. 상태별 화면 동작

### BEFORE_START

- 참가자 투자 불가
- 참가자 화면에 “게임 시작 전입니다.” 표시
- 관리자만 연차 시작 가능

### SALARY_PAID

- 참가자 현금 증가
- 투자 버튼은 아직 비활성화 가능

### INVESTING

- 참가자 투자 가능
- 관리자 투자 마감 가능
- 전광판 타이머 표시

### INVEST_CLOSED

- 참가자 투자 불가
- 관리자 정산 가능

### SETTLED

- 참가자 결과 확인 가능
- 관리자 다음 연차 이동 가능

### REALTIME_ROUND

- 4년차 전용
- 참가자 투자 가능
- 실시간 그래프 표시
- 전광판 실시간 순위 표시

### PAUSED

- 투자 불가
- 타이머 정지
- 그래프 갱신 정지
- 참가자 화면에 일시정지 공지 표시

### FINISHED

- 투자 불가
- 최종 엔딩 표시
